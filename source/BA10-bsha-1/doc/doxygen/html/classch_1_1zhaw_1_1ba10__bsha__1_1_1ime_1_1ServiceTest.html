Detailed Description

Activity we used to test the proper operation of our DetectionService. Will not be needed for a possible deployment but we kept it as an entry point to our Application during development.

Based on the Remote Service Controller and Remote Service Binding code samples from the API Demos example included in the Android SDK (under App/Service).

                @Override
                public void handleMessage(Message msg) {
            switch (msg.what) {
                case CHARS_RESULT_MSG:
                        ArrayList<Character> chars = (ArrayList<Character>) msg.obj;
                        if (chars.size() > 0) {
                                txtViewResult.setText("Received from service: " + chars.get(0).toString());
                        } else {
                                txtViewResult.setText("Received from service: nothing");
                        }
                    break;
                case CHAR_RESULT_MSG:
                        char recogn_char = (char) msg.arg1;
                        float prob = (float) (msg.arg2 / 1000000.0);
                        txtViewResult.setText(txtViewResult.getText().toString() + recogn_char + "(" + prob + ")");
                    break;
                default:
                    super.handleMessage(msg);
            }
        }
